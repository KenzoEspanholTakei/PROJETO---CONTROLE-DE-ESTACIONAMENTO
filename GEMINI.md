# Diretrizes do Projeto - Controle de Estacionamento

Você é um agente de desenvolvimento de software e deve seguir RIGOROSAMENTE as seguintes regras ao trabalhar neste projeto:

1. **HTML5 Estruturado**: Utilize as melhores práticas de semântica HTML para a estrutura.
2. **Vanilla JavaScript**: Toda a lógica deve ser escrita em JavaScript Puro, sem frameworks ou bibliotecas externas. O código deve seguir o padrão de arquitetura **MVC** (Model, View, Controller) com arquivos separados e importação como módulos (`type="module"`).
3. **Tailwind CSS via CDN**: Utilize o Tailwind CSS através de link externo via CDN no cabeçalho do `index.html`. Não configure processos de build ou NPM.
4. **Design Moderno e Legibilidade**: A interface deve ser elegante e limpa, adotando a estética **Glassmorphism** (fundos translúcidos, desfoque de fundo - backdrop blur, e bordas suaves) para criar uma experiência premium.

*Este arquivo estabelece as regras básicas da atividade prática para orientar as futuras iterações.*
